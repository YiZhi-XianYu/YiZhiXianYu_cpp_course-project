# 🗡️ Tiny RPG (南开大学26 C++ 课程大作业)

本项目是 **南开大学26级C++课程的大作业**。这是一款基于 **C++** 核心驱动的 2D 像素风 回合制 RPG 游戏，通过跨平台 WebAssembly (WASM) 技术与前端结合，并最终使用 **Node.js 打包为独立桌面应用程序**。无需任何繁琐的配置，开箱即玩！

## 🎮 快速游玩 (零环境配置)

为了提供最流畅的体验，我已经使用 Node.js 将游戏打包成了独立的运行程序，**玩家无需自带浏览器，更无需配置任何开发或运行环境**（不需要安装 Node.js、C++ 环境或任何服务器）：

1. 前往本仓库的 **Releases** 页面下载最新的游戏压缩包 (`.zip`)。
2. 将压缩包解压到本地电脑的任意文件夹中。
3. 双击运行其中的 `YiZhiXianYuGame.exe` 应用程序。
4. 游戏将在独立的窗口中原生启动，即刻开始你的地牢冒险！

---

## ✨ 核心玩法 (Gameplay)

- **动作战斗机制**：玩家可以通过算好的走位躲避敌人的攻击，并使用近战或远程武器进行反击。
- **索敌与仇恨系统**：游戏内置了智能的怪物 AI，当你踏入怪物的领地时，它们会被激怒并向你发起无情的追击。
- **沉浸式动态音效**：
  - **环境音**：昼夜环境音效，以及地牢内的氛围背景乐。
  - **专属 BGM**：昼夜更替、地图切换时，BGM 会智能切换。
  - **拟真 SFX**：包含石板路脚步声、挥剑风声、弓箭破空声、受击反馈及死亡音效。
- **平滑运镜**：底层 C++ 实现的 `CameraController` 提供丝滑的镜头跟随，确保主角始终处于视觉焦点。

## 🦸 角色与怪物

### 🧑‍🚀 玩家 (Soldier)
- **职业设定**：身披重甲、精通多种武器的精锐士兵。
- **移动方式**：通过键盘 W A S D 或 上 左 下 右 按照回合移动。
- **攻击方式**：
  - **近战挥砍**：使用长剑进行三段式连击，对近身敌人造成毁灭性打击，对应键盘 J 或鼠标左键。
  - **远程射击**：挽弓搭箭，在安全距离外对敌人进行消耗对应键盘 J 或鼠标左键。
  - **强大技能**：两种职业各有两个强大的技能，对应键盘 E 和 Q 。
- **核心驱动**：由 C++ `PlayerController` & `CharacterRole` 类管理，具有待机 (Idle)、行走 (Walk)、攻击 (Attack 1/2/3)、受击 (Hurt) 和死亡 (Death) 等全套像素动画状态机与碰撞判定。

### 👹 怪物 (Orc 兽人)
- **背景设定**：盘踞在地牢深处的野蛮兽人，手持重型砍刀。
- **AI 行为**：由 C++ `MonsterController` 驱动。具备巡逻、索敌与攻击逻辑。一旦玩家进入其警戒范围，兽人会立刻中断巡逻，向玩家冲锋。
- **战斗反馈**：拥有专属的刀劈音效、受击硬直动画以及死亡倒地（Death Spin）演出。

## 🗺️ 游戏地图 (Maps)

游戏世界使用 **Tiled Map Editor** (`.tmx`) 精心搭建，包含不同难度的关卡：

- **Demonstration (演示关卡/主页)**：用于展示游戏主界面与昼夜更替效果的初始场景。
- **💀 地牢一层 (地牢1)**：初窥门径。地形相对开阔，兽人分布零散，适合玩家熟悉移动与攻击节奏。
- **💀 地牢二层 (地牢2)**：危机四伏。视野昏暗，路线开始变得错综复杂，巡逻的兽人数量增加，玩家需要利用走位进行拉扯。
- **💀 地牢三层 (地牢3)**：深渊死斗。强大的怪物与广阔的地形，考验玩家对近战与远程攻击的合理使用。

## 🛠️ 技术架构

- **后端核心**: C++ (处理实体状态机、AI、物理碰撞与摄像机逻辑)。
- **跨平台桥接**: Emscripten (将 C++ 编译为 WASM 文件 `game_core.wasm` 并生成胶水代码)。
- **前端展现**: HTML5 Canvas / CSS / JavaScript (负责图形渲染与用户输入监听)。
- **桌面端打包**: 基于 Node.js 生态构建，将前端网页与 WASM 模块直接封装为 `.exe` 独立桌面应用。
